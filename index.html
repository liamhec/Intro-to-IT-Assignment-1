Project Idea

Overview

I think it would be interesting to develop a small-scale robot that has the ability to move on its own. Whether this bot has the form as a car or an actual robot, it should be able to move on its own without crashing into obstacles. For this project a raspberry pi would probably be used, due to its small factor, which would allow it to easily fit into any shell. The software would need to be developed for this bot, and the ability to interpret data being inputted from sensors or cameras, while moving forward.

Motivation

I believe this project will be interesting due to the rapid pace of automated cars, vacuum cleaners and such rising and being sold on the market. This project would be on a smaller scale compared to these products, but the knowledge and experience gained would be useful in this new area of the I.T industry. With artificial intelligence (A.I) becoming a large sector of the I.T industry, this project would be a great start and look into what A.I is all about. Its expected that A.I could in fact become a $2.2 trillion opportunity by 2030 (ABC), so this industry is only expected to grow.

Description

This project involves developing an artificial intelligence (A.I), that can control a toy car without human input. The vehicle should be able to interpret data from a camera and make decisions such as the best way to avoid objects. A raspberry pi will have a program loaded onto it, that will ultimately control the cars actions. The program will be developed using java with a combination of two open source code libraries. The first open source code is called OpenCV. This open source code can be trained to detect objects through the use of a camera. This is essential as this is what will first identify objects and locate where they are. OpenCV is a good choice for this project as it works in java, but it also works with the raspberry pi camera in order to actually be able to identify objects. The other open source code that will accompany OpenCV is TensorFlow. TensorFlow can be used to actually make decisions based upon what OpenCV has identified. This might be one of the most important pieces to this project, as it will decide what to do when there is an object, and also it will control the basic functions such as steering and speed. This project will create essentially an artificial intelligence that could be used on other small projects and implemented as a solution to small objects that need the capability to control their own actions. This project will grant a small car the ability to move around without crashing into other objects, which could have a number of uses. Currently on this scale, vacuum cleaners are able to move around on their own and not crash into other objects, there are also lawn mowers that take this idea, and are to mow lawn by themselves without crashing. On a larger scale, actual cars are being developed to drive on their own, which means they need the drive without crashing into other cars, and pedestrians, while also steering and at high speeds. Creating a project like this on a smaller-scale demonstrates the power of artificial intelligence, and the capabilities of technology. This project should allow for a toy car to have the ability to have little to no human intervention. It should be able to steer on its own and select a reasonable speed to be travelling at. The car should also have the ability to identify any objects around it, especially in front of it, and make a decision on what to do in real time. Whether the decision to be stop, drive around it, or turn around, all these decisions should be made on its own.

Tools and Technologies

The artificial intelligence (A.I) will need to be programmed. A good language to use with raspberry pi would be JavaScript. This will allow for some open source software libraries to be used, to reduce the complexity of the project. For example, OpenCv can be used to detect objects, for this bot to not walk into objects. The project will need a shell, such as a car or a something similar, and a raspberry pi to control the vehicle, as well as a sensor or camera for data input, to identified objects and avoid them.

Skills required

Knowledge in Java will be needed in order to program this autonomous vehicle, and for it to be able to avoid obstacles. This will need to be accompanied with knowledge on how to implement this onto a raspberry pi in order to control the vehicle. Also, some knowledge in open source code will be needed in order to also implement this. The main part of special hardware needed in a pi camera, in order to see objects, and therefore know how to react. Raspberry pi and the pi camera are readily available online, as well as the open source code, being able to program this bot shouldn’t be too difficult to find people who have some knowledge in java, and we can build upon this.

Outcome

If this project is successful, there will be a small car with the ability to drive on its own, and avoid objects, without crashing into them. The software could be applied to other small objects, possibly with some small amount of tweaking. The impact of this project would be more upon the team who are working on it. It will expose us to the rapidly growing sector of the I.T industry, artificial intelligence. This project would provide a somewhat decent idea of the type of work goes into developing an A.I and how it can be implemented in real world problems, as well as the process in which is undertaken in order to implement something like this.
